# Universo de Juego de Tronos

Aplicación web simple desarrollada con HTML, CSS y JavaScript.

El sitio consume información desde ThronesAPI y presenta personajes de Juego de Tronos en una página propia.

## Páginas

- Página principal
- Página personajes

## Funciones

- Visualización de integrantes del grupo
- Consumo de API con JavaScript
- Buscador de personajes
- Detalles de personajes
- Preguntas para encontrar un personaje relacionado con las respuestas
